Full Backpack Basic Theory

  • To ensure that each item is only added once. The inner loop order of the 01 backpack is from big to small.
  • The items in the full backpack can be added multiple times, so we must traverse from small to big.
  • The order of inner and outer loop can be reversed.
  • If we solve for the number of combinations, the outer loop traverses items, the inner loop traverses capacity.
  • If we solve for the number of permutations, the outer loop traverses capacity, the inner loop traverses items.

518. Coin Change II

Question Link

class Solution {
    public int change(int amount, int[] coins) {
        int[] dp = new int[amount + 1];
        dp[0] = 1;
        for(int i = 0; i < coins.length; i++){
            for(int j = coins[i]; j <= amount; j++)
                dp[j] += dp[j-coins[i]];
        }   
        return dp[amount];
    }
}
  • dp[j]: The number of currency combinations that make up the total amount j.
  • When we solve for the number of ways to fill the backpack, the recursive formula is:
    • dp[j] += dp[j - coins[i]]
  • dp[0] must be initialized to 1. If dp[0]=0, all the value of dp will be 0.

377. Combination Sum IV

Question Link

class Solution {
    public int combinationSum4(int[] nums, int target) {
        int[] dp = new int[target + 1];
        dp[0] = 1;
        for(int i = 0; i <= target; i++){
            for(int j = 0; j < nums.length; j++){
                if(i >= nums[j]) dp[i] += dp[i - nums[j]];
            }
        }
        return dp[target];
    }
}

  • Note that different sequences are counted as different combinations. So in this question, what we are essentially looking for is the permutation.

  • If we solve for the number of permutations, the outer loop traverses capacity, the inner loop traverses items.

    • For example, if the outer loop traverses items, when we calculate the dp[4], the result combination includes the {1, 3}, not {3, 1}. Cause 3 only presents behind the 1

  • dp[j]: The number of nums permutations that make up the target
