Spring CLI - Windows Scoop Installation

2023-08-16 1 min read

💡Introduction

Scoop is a command-line installer for Windows. For actual users, it is very similar to the package management tool.

🥄Install the Scoop

Allowed execution of local scripts:

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Install the Scoop

irm get.scoop.sh | iex

Install aria2 for multi-thread download acceleration:

scoop install aria2

Test whether the installation is successful:

scoop help

🌱Install the Spring Boot CLI

scoop bucket add extras
scoop install springboot

📝Common Command

Command Description
search 搜索软件名
install 安装软件
update 更新软件
status 查看软件状态
uninstall 卸载软件
info 查看软件详情
home 打开软件主页

Install the specified spring verison:

scoop install springboot@2.5.6
